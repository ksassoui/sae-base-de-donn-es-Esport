# SAE Base de Données - eSport

## Contexte

Projet réalisé dans le cadre de la SAE S2.04 Exploitation d'une Base de Données.

Le sujet attribué à mon groupe portait sur l'univers de l'eSport.

---

## Besoin

Concevoir une base de données relationnelle permettant de gérer :

- les équipes
- les joueurs
- les tournois
- les matchs
- les résultats

---

## Technologies utilisées

- PostgreSQL
- SQL
- Draw.io
- LibreOffice

---

## Compétences mobilisées

### Gérer des données

- Modélisation de données
- Construction d'un MLD
- Création des relations

### Exploiter les données

- Requêtes SQL
- Analyse de données
- Création de graphiques

### Conduire un projet

- Travail en équipe
- Répartition des tâches

---

## Réalisations

- Conception du modèle logique de données
- Création du script SQL
- Insertion des données
- Création de requêtes SQL
- Analyse statistique des données

---

## Difficultés rencontrées

- Définition des relations entre les tables
- Respect de la cohérence des données
- Construction de requêtes complexes

---

## Résultat

La base permet de représenter un environnement eSport complet et d'extraire des indicateurs grâce aux requêtes SQL.

---

## Si c'était à refaire

J'ajouterais davantage de statistiques et une interface graphique permettant de consulter les données plus facilement.
